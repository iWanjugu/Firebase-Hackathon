What's the event about?

Are you looking to build your own real-time apps, join us and together we can build some of your very own apps from the ground up using Firebase. Firebase allows you to build real time mobile and web apps in a matter of minutes using client-side code and a powerful API.

Technology

Firebase was built to make the real-time technology that power Envolve's caht product accessible to developers, so that they can build their own powerful applications. It incorporates years of learning about how to handle real-time data on the web and provides a solution to many issues faced by developers.

Join Us

This event takes place at Nairobi Garage on Saturday 12th September from 9:00AM to 4:00PM. You will have to bring a laptop but it is a completely free event! We are excited to see you attend and build your very own apps!
